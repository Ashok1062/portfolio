What is coding in computer:

Coding,also known as programming, is the process of writing instruction (code)that a computer can understand and follow to perform specific tasks, enabling the creation of software,websites ,apps,and more.

Examples:

  • Web Development: HTML,CSS,and JavaScript are used to create and style websites

  • Software Development: Language like java and C++ are used to bulid application and operating systems.

  • Data Analysis: Language like python and R are used to analyze and manipulate data
